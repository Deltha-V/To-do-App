# To-Do List App (Flask + SQLite)

Aplicación web de lista de tareas desarrollada en Python usando Flask y SQLite.

## Features
- Crear tareas
- Listar tareas
- Marcar tareas como hechas
- Editar tareas
- Eliminar tareas
- Persistencia con SQLite

## Tecnologías
- Python
- Flask
- SQLite
- HTML / CSS (Jinja2 templates)
